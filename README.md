# Typing Game Server

A server handle all the tasks needed for https://github.com/Youi-4/typingGameClient.

## Features
- User authentication
- CRUD operations
- JWT authorization
- Postgresql database

## Tech Stack
- Node.js
- Express
- Postgresql

## How to Run
1. Clone this repo 
2. Run npm install
3. Create .env file with values corresponding with your Postgresql database
4. Run "npm run dev"
